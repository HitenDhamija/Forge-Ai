# API Reference

## Endpoints

| Method | Path                                                                                           | Description                                                                                                                                                                                                                   |
|--------|--------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| GET     | /dalle                                                                                          | Fetches content generated by DALL-E based on user input.                                                                                                                                                                       |
| POST    | /upload-image                                                                                     | Uploads an image to the server and returns a URL where the uploaded image can be accessed.                                                                                                                                 |
| GET     | /users/:id                                                                                       | Retrieves details of a specific user identified by `:id`.                                                                                                                                                                         |
| DELETE  | /users/:id                                                                                       | Deletes a user with the specified ID from the database.                                                                                                                                                                          |
| POST    | /orders                                                                                          | Creates an order for a product and returns the newly created order's details.                                                                                                                                                   |
| GET     | /products                                                                                        | Retrieves all products available in the store.                                                                                                                                                                                |
| PUT     | /products/:id                                                                                     | Updates the details of a specific product identified by `:id`.                                                                                                                                                                      |
| DELETE  | /products/:id                                                                                     | Deletes a product with the specified ID from the database.                                                                                                                                                                        |
| GET     | /orders                                                                                          | Retrieves all orders made by users.                                                                                                                                                                                        |
| POST    | /auth/login                                                                               | Authenticates a user and returns an authentication token upon successful login.                                                                                                                                                   |
| DELETE  | /auth/logout                                                                                  | Logs out a user and clears their session data.                                                                                                                                                                                  |
| GET     | /users/me                                                                                        | Retrieves the authenticated user's details if they are logged in.                                                                                                                                                                 |

## Request/Response Format

### POST /upload-image
**Request Payload:**
```json
{
  "image": "<base64 encoded image>"
}
```

**Response Payload (Success):**
```json
{
  "url": "https://server.com/image.png"
}
```

**Response Payload (Error - Image Upload Failed):**
```json
{
  "error": "Image upload failed. Please try again."
}
```

### GET /users/:id
**Request Payload:**
None

**Response Payload (Success):**
```json
{
  "id": "<user_id>",
  "name": "<user_name>",
  "email": "<user_email>"
}
```

**Response Payload (Error - User Not Found):**
```json
{
  "error": "User not found."
}
```

## Error Handling

Errors are handled using HTTP status codes. Common error messages include:

- **401 Unauthorized**: When authentication fails, e.g., invalid credentials.
- **403 Forbidden**: When the user is logged in but does not have permission to access a resource.
- **500 Internal Server Error**: When an unexpected internal server error occurs.

For detailed error handling logic, refer to the respective route files and handler names:

- `/src/routes/dalle.routes.js`
- `/server/index.js`

These files contain the actual implementation of each endpoint.
